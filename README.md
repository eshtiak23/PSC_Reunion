# Reunion 2026 - One Bond One Memory

A clean, modern reunion management website with Google Sheets integration.

## Pages
- **Home** - Hero, countdown, stats, recent payments, members preview
- **Members** - Hardcoded member list with payment status
- **Payments** - Live data from Google Sheets (Name, Date & Time, Amount, Method)
- **Memories** - Gallery placeholders

## Setup
1. Edit `config.js` with your Google Spreadsheet ID and Apps Script URL
2. Deploy Google Apps Script as Web App
3. Open `index.html` or deploy to GitHub Pages

## Tech
- HTML5, CSS3, Vanilla JavaScript
- Google Sheets API via Apps Script
- Light glassmorphism theme (pink, purple, blue)
